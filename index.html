How do you define an AngularJS application?

Create an angular module and then attach the module to the html tag of your index.html.

What is a controller?

A controller sits between the view and the data and is there to bind the data to the view

What does ng-model do?

ng-model is an Angular directive that enables HTML inputs to be bound to a controller's scope

What is $scope?

What is dependency injection?

Had to research this as I definitely do not know this off the top of my head... My research shows that it is the act of injecting methods and objects that are needed by angular modules/controllers/factories/services/components/directives/filters/etc

What is two-way binding?

Using $scope (or any other Controller As alias) and ng-model to allow data to pass from the view to the controller and vice versa

What is 'controller as' syntax?

It's a way of providing a unique name to the scope of the controller in question

What is MV* architecture?

Stands for Model, View, Whatever

What is the difference between a service and a factory?

A service is an object and a factory is a function
